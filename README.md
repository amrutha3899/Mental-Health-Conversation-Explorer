**Mental Health Conversation Explorer**

**Overview**

An interactive AI-inspired data exploration tool built using Streamlit to analyze simulated mental health conversations.

This project enables users to explore structured conversational data through dynamic filtering and intuitive navigation, helping uncover patterns in human communication and emotional expression.


**Problem**

Conversational datasets—especially in sensitive domains like mental health—are often unstructured and difficult to explore. Traditional analysis methods lack interactivity and fail to capture the nuance in conversations.


**Solution**

Built a lightweight, interactive application that allows users to:

1. Filter conversations by category, sub-category, and type
2. Navigate through conversations in a structured format
3. Explore patterns in emotional tone and communication style


**Tech Stack**

1. Python<br>
2. Streamlit<br>
3. Pandas<br>


**Key Features**

1. Dynamic filtering of conversational data
2. Conversation viewer with step-by-step navigation
3. Cached data loading for performance optimization
4. Structured taxonomy-based exploration


**Dataset**

The dataset consists of simulated mental health conversations with:

1. Categorized stress types
2. Conversation patterns
3. Sentiment indicators
4. Empathy and listening cues

_(Note: Dataset is synthetic and used for exploration purposes only.)_
